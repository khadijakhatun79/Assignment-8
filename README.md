# 🌞 SunCart – Summer Essentials Store

## 📌 Project Overview
SunCart is a modern summer-themed eCommerce web application where users can explore and purchase seasonal products like sunglasses, outfits, skincare items, and beach accessories. The platform provides a smooth user experience with authentication, protected routes, and responsive design.

---

## 🎯 Purpose
The main purpose of this project is to build a fully functional eCommerce website using modern web technologies. It demonstrates user authentication, product browsing, and responsive UI design.

---

## 🔗 Live Website
👉 https://assignment-8-rho-two.vercel.app

---

## 🚀 Key Features

- 🔐 User Authentication (Login & Register with Email/Password & Google)
- 🔒 Protected Routes (Product Details page only for logged-in users)
- 🛍️ Product Listing from JSON data
- 📄 Product Details Page with full information
- 👤 My Profile Page (User info display)
- ✏️ Update Profile (Name & Image update)
- 🎬 Animated Hero Section (Swiper + Animate.css / Lottie)
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 🌐 Smooth Routing (Next.js App Router)
- ⚡ Fast Deployment (Vercel)

---

## 🧰 Technologies Used

- ⚛️ Next.js
- 🎨 Tailwind CSS
